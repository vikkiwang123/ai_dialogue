# 图标文件说明

本目录需要包含以下三个PNG图标文件：

- `icon16.png` - 16x16像素
- `icon48.png` - 48x48像素  
- `icon128.png` - 128x128像素

## 生成图标

### 方法1：使用Python脚本

```bash
pip install Pillow
python generate-icons.py
```

### 方法2：使用网页工具

在浏览器中打开 `create-icons.html`，点击"生成图标"按钮。

### 方法3：手动创建

使用任何图像编辑工具创建16x16, 48x48, 128x128像素的PNG图标。

图标建议：
- 使用蓝色主题（#2196f3）
- 包含AI或对话相关的视觉元素
- 简洁清晰，在小尺寸下也能识别

## 临时解决方案

如果暂时没有图标文件，可以：

1. 从网上下载任意16x16, 48x48, 128x128的PNG图片
2. 重命名为对应的文件名
3. 放入此目录

插件可以正常加载，只是图标可能不太合适。

# 快速开始指南

## 5 分钟快速安装

### 步骤 1：生成图标（1 分钟）

**最简单的方法**：
1. 双击打开 `assets/icons/download-icons.html`
2. 点击"下载所有图标"按钮
3. 将下载的 3 个 PNG 文件复制到 `assets/icons/` 目录

### 步骤 2：安装插件（1 分钟）

1. 打开 Chrome 或 Edge 浏览器
2. 地址栏输入：`chrome://extensions/`（Edge 用 `edge://extensions/`）
3. 右上角开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹（`ai_dialogue`）
6. ✅ 完成！

### 步骤 3：配置 AI 总结（2 分钟）

1. 右键插件图标 → **选项**
2. 在「大模型配置」中选择供应商（推荐 Moonshot / Kimi）
3. 输入你的 API Key
4. 点击「保存配置」→「测试连接」确认可用
5. ✅ 配置完成！

### 步骤 4：开始使用（1 分钟）

1. 访问 [ChatGPT](https://chatgpt.com)、[Claude](https://claude.ai) 或任何支持的 AI 平台
2. 正常对话，插件会自动记录
3. 点击浏览器工具栏的插件图标 → **侧边栏**在右侧打开
4. 切换到「🧠 总结」标签页，点击「生成总结」
5. 🎉 AI 总结会以流式打字效果逐字呈现！

> **💡 提示**：侧边栏打开后会固定在浏览器右侧，无需反复点击图标。

## 验证安装

安装成功后，您应该看到：

- ✅ 浏览器工具栏出现插件图标
- ✅ 点击图标后侧边栏在右侧打开
- ✅ 访问 AI 平台时，控制台显示"AI对话监控助手 已加载"
- ✅ 对话后侧边栏中统计数字更新

## 常见问题

### Q: 图标显示为灰色或空白？
A: 需要先生成图标文件，参考步骤 1。

### Q: 点击图标没反应？
A: 确保浏览器支持 Side Panel API（Chrome 114+ / Edge 114+）。如果版本较低，更新浏览器即可。

### Q: 无法监控消息？
A: 检查：
- 当前页面是否是支持的 AI 平台
- 插件设置中是否启用了对应平台
- 在扩展管理页面点击「重新加载」后刷新页面

### Q: AI 总结报错？
A: 检查：
- 是否已在设置页面配置 API Key
- API Key 是否有效（使用「测试连接」按钮检查）
- 当天是否有对话记录

### Q: 数据在哪里？
A: 所有数据存储在浏览器本地（chrome.storage.local），可以通过：
- 侧边栏查看统计和消息
- 设置页面查看详细数据、导出或清除
